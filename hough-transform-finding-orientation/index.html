Hough Transform for Finding Orientation

The Hough Transform is a classical computer vision technique used to detect geometric shapes such as lines, circles, and ellipses. In this project, we use it specifically to determine the orientation of an object or defect by detecting the dominant line direction.

Why Use the Hough Transform?

The edges or boundary pixels of an object often form elongated patterns. By finding the strongest straight line that aligns with these pixels, we can determine the major orientation. The Hough Transform allows us to do this by mapping pixels into a parameter space and accumulating evidence for possible lines.

Hough Transform Step-by-Step Approach

The Hough Transform starts by creating multiple lines in many directions for a single pixel inside the defect cluster. For each direction, it computes an angle (θ) and a distance from the origin (ρ). Each (ρ, θ) pair receives a vote in the accumulator. When multiple boundary pixels produce the same (ρ, θ), the votes accumulate. After all pixels are processed, the pair with the highest votes corresponds to the best-fitting line. The angle θ gives the main orientation of the defect.

Hough Transform — Step-by-Step Explanation

In the image above, the detailed steps of the process are visualized. The red pixel points represent the defect boundary.

  • For each boundary pixel, the algorithm draws straight lines in multiple directions (0°–180°). Each line corresponds to a (ρ, θ) pair stored in the accumulator (Hough space).
  • When a green line is drawn, its (ρ, θ) pair is stored and receives its first vote.
  • The same process is repeated for the purple line, blue line, and others, each contributing votes to the accumulator.
  • After covering all directions for one pixel, the algorithm moves to the next boundary pixel and repeats the process, updating the accumulator.
  • If the same blue line appears for another pixel, the accumulator receives another vote for the same (ρ, θ) pair.

This continues for all boundary pixels. The (ρ, θ) pair with the highest vote count represents the most consistent line across the defect boundary. This line provides the main orientation relative to the x-axis.

Example: Hough Transform of a Straight Line

In this example, the boundary of a straight line is extracted. Each boundary point maps to a curve in Hough space. Curves generated by points lying on the same straight line intersect at a common (ρ, θ), forming a peak. The θ value at this intersection corresponds to the detected orientation.

Orientation Extraction

After identifying the strongest (ρ, θ) peak:

  • θ gives the line orientation.
  • Horizontal lines → θ = 0°
  • Vertical lines → θ = 90°
  • Other lines → orientation depends on detected θ
